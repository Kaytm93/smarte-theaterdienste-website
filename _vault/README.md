# Project Vault – Smarte Theaterdienste

Obsidian-Vault als Projektplaner. Im Repo versioniert (`_vault/.obsidian/workspace.*`
und `cache` sind in `.gitignore`).

## Struktur

| Ordner | Inhalt |
|---|---|
| `01_Konzept/` | Projektziel, Zielgruppen, Tonality, Inhalts-Strategie |
| `02_Architektur/` | Tech-Stack, DB-ER-Diagramm, i18n-Strategie, ADRs |
| `03_Komponenten/` | Komponenten-Hierarchie, Animationsplan |
| `04_Content/` | Texte, Assets, Lizenzen |
| `05_Timeline/` | Milestones M1–M8 mit Status |
| `99_Notes/` | Entscheidungsprotokolle, Loose Notes |

## Quick Open

In Obsidian → „Open folder as vault" → `_vault/`.
